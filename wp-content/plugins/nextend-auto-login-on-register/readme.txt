The plugin will automatically log in your new visitor after the register process.

== Description ==

Check the [DEMO](http://www.nextendweb.com/wp-login.php?action=register) on our site.



If you like our stuff donate a like to our [Facebook page](https://www.facebook.com/nextendweb) or follow us on [Twitter](https://twitter.com/nextendweb)!

#### Usage

Simply activate the plugin and it will work out of the box.

== Installation ==

1.  Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
